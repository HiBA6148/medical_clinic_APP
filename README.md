# Medical Clinic Management System 🏥
A Java-based desktop application developed for the L2 ACAD curriculum at USTHB. The platform digitizes clinic operations, providing a secure portal to manage patients, doctors, and appointments through a modern interface linked to an Oracle Database.

Key Contributions

Frontend & UI/UX: Designed a modern "glassmorphism" login and dashboard using Java Swing.

Database Integration: Connected the interface to Oracle SQL using JDBC (ojdbc11) for secure data persistence.

Collaboration: Integrated backend and frontend components using Git/GitHub for team synchronization.
